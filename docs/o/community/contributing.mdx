# Community: Contributing to Nanoservice-ts

Nanoservice-ts is an open-source project, and we welcome contributions from the community! Whether you want to report a bug, suggest a feature, submit a pull request for code or documentation, or help answer questions, your involvement is valuable.

## How to Contribute

-   **Reporting Bugs:** If you find a bug, please open an issue on our [GitHub repository](https://github.com/deskree-inc/nanoservice-ts/issues), providing as much detail as possible, including steps to reproduce, expected behavior, and actual behavior.
-   **Feature Requests:** For new features or enhancements, please open an issue with a clear description of the proposed feature and its use case.
-   **Pull Requests:** We welcome pull requests for bug fixes, new features, and documentation improvements. Please follow our coding style and ensure your changes pass any existing tests. For significant changes, itâ€™s a good idea to discuss them in an issue first.
-   **Documentation:** Help us improve the documentation by reporting errors, suggesting clarifications, or contributing new content.
-   **Community Support:** Help other users by answering questions on GitHub Discussions, Stack Overflow, or other community channels.
